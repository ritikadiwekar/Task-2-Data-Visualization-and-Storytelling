# Task 2 - Data Visualization and Storytelling

## Project Title
Superstore Sales & Profitability Analysis Dashboard

## Tool Used
- Power BI

## Dataset
- Superstore Dataset

## Dashboard Pages

### 1. Executive Overview
- Total Sales
- Total Profit
- Total Orders
- Total Customers
- Profit Margin
- Sales Trend Analysis
- Region & State Performance

### 2. Profitability Analysis
- Profit by Category
- Profit by Region
- Profit by Sub-Category
- Discount Impact on Profitability
- Loss Order Analysis

### 3. Customer & Product Insights
- Average Sales per Customer
- Average Order Value
- Average Quantity per Order
- Top Customers by Revenue
- Sales by Ship Mode
- Sales by Sub-Category

## Key Insights
- Total Sales reached 2.30M.
- Total Profit reached 286.40K.
- Profit Margin was 12.47%.
- Technology category generated the highest sales.
- West region was the top-performing region.
- California contributed the highest sales.
- Phones and Chairs were the best-selling sub-categories.
- Higher discounts negatively impacted profitability in some sub-categories.
